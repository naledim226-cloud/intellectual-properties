How to create the Google Form

  1. Create a Google Form (forms.google.com) titled "Lead Submission"
  2. Add fields: Agent name, Agent email, Lead name, Phone, Email, Property type, Budget, Suburb, Notes
  3. Connect responses to a Google Sheet (Responses → Link to Sheets)
  4. Copy the Embed URL (⋮ → Embed) and replace the iframe src in this file
